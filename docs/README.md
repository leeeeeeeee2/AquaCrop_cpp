# AquaCrop-cpp Documentation

Overview of the documentation structure and how to contribute to docs.

Contents:
- API: docs/api.md
- Design: docs/design.md
- Guides: docs/guides/
- Tutorials and references: docs/guides/tutorials/
- C++ full documentation: docs/cpp_aquacrop_full_documentation.md

Contributing to docs:
- Follow the same contribution process as code changes.
- Ensure API references stay in sync with the public headers.
- Run any included docs build steps if provided by the project.
